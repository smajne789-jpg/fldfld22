# CryptoBot Game Bot

Красивый Telegram-бот с меню, профилем, пополнениями через CryptoBot, автовыводом через CryptoBot, внутренними чеками, резервом и играми на реальных Telegram dice.

## Возможности

- `/start` открывает главное меню, дальше пользователь ходит по кнопкам и не пишет старт повторно.
- Профиль: баланс, пополнения, выводы, статистика игр, кнопки пополнения и вывода.
- Пополнение от `0.30 $` через Crypto Pay `createInvoice`.
- Вывод от `1.70 $`; для вывода нужно иметь депозитов минимум на `1.00 $`.
- Автовывод через Crypto Pay `transfer`; если автовывод отключен, заявка уходит администраторам в личные сообщения.
- Резерв показывается в меню и меняется в админ-панели.
- Игры:
  - Куб 18+: два кубика, произведение больше 18, выигрыш `x5`.
  - Куб нечет: один кубик, выпадает 1/3/5, выигрыш `x2`.
  - Куб чет: один кубик, выпадает 2/4/6, выигрыш `x2`.
  - Куб 7: два кубика, сумма ровно 7, выигрыш `x4`.
  - Куб 7+: два кубика, сумма больше 7, выигрыш `x4`.
- Админ-панель:
  - изменение резерва;
  - создание внутренних чеков-ссылок с суммой, количеством активаций и требуемой суммой депозитов;
  - включение/отключение автовывода;
  - статистика проекта;
  - approve/decline заявок на вывод.

## Установка

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
copy .env.example .env
```

Заполните `.env`:

- `BOT_TOKEN` - токен Telegram-бота от BotFather.
- `CRYPTO_PAY_TOKEN` - токен приложения Crypto Pay.
- `ADMIN_IDS` - Telegram ID админов через запятую.
- `PAYOUT_ASSET` - актив для выводов, по умолчанию `USDT`.

## Запуск

```bash
python -m app.main
```

База SQLite создастся автоматически в `data/bot.sqlite3`.

## Важные замечания

- Пополнения создаются как fiat-инвойсы в USD. Для вывода бот отправляет `PAYOUT_ASSET`; при `USDT` сумма вывода считается как USD ~= USDT.
- Внутренние чеки начисляют деньги на баланс бота по deep-link. Это сделано специально под сценарий "сумма + число активаций + требуемые депозиты".
- Для автовывода через CryptoBot пользователь должен иметь возможность получать transfers от вашего Crypto Pay приложения.
- Бот использует polling и сам периодически проверяет неоплаченные инвойсы. Также пользователь может нажать кнопку "Проверить оплату".

Документация Crypto Pay API: https://help.crypt.bot/crypto-pay-api/
